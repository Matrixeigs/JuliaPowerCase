# I/O Formats

JuliaPowerCase supports multiple input/output formats for power system data.

---

## MATPOWER Import

### `load_matpower`

Parse MATPOWER `.m` case files.

```julia
load_matpower(filepath::String; T=Float64) -> PowerCaseData{AC, T}
```

**Parameters:**
- `filepath` — Path to `.m` file
- `T` — Numeric type (default: `Float64`)

**Supported Fields:**
- `mpc.baseMVA`
- `mpc.bus`
- `mpc.gen`
- `mpc.branch`
- `mpc.gencost`

**Example:**
```julia
using JuliaPowerCase

# Load IEEE 14-bus case
jpc = load_matpower("case14.m")

println("Base MVA: ", jpc.base_mva)
println("Buses: ", nrows(jpc.bus))
println("Generators: ", nrows(jpc.gen))
println("Branches: ", nrows(jpc.branch))
```

### MATPOWER File Format

```matlab
function mpc = case5
mpc.version = '2';
mpc.baseMVA = 100;

%% bus data
%   bus_i type Pd    Qd    Gs Bs area Vm    Va  baseKV zone Vmax Vmin
mpc.bus = [
    1     3    0     0     0  0  1    1.05  0   110    1    1.1  0.9;
    2     1    100   30    0  0  1    1.0   0   110    1    1.1  0.9;
    3     1    80    20    0  0  1    1.0   0   110    1    1.1  0.9;
];

%% generator data
%   bus Pg   Qg   Qmax Qmin Vg   mBase status Pmax Pmin
mpc.gen = [
    1   150  0    100  -100 1.05 100   1      200  0;
];

%% branch data
%   fbus tbus r      x     b    rateA rateB rateC ratio angle status
mpc.branch = [
    1    2    0.01   0.1   0.02 100   100   100   0     0     1;
    2    3    0.02   0.2   0.03 100   100   100   0     0     1;
];
```

## Julia Case Files

### `save_julia_case`

Save system as Julia source file.

```julia
save_julia_case(filepath::String, sys::PowerSystem{K})
save_julia_case(filepath::String, jpc::PowerCaseData{K})
```

**Example:**
```julia
sys = case14()
save_julia_case("my_case14.jl", sys)
```

### `load_julia_case`

Load system from Julia source file.

```julia
load_julia_case(filepath::String) -> PowerSystem
```

**Example:**
```julia
sys = load_julia_case("my_case14.jl")
```

### Generated File Format

```julia
# Auto-generated by JuliaPowerCase
function my_case()
    sys = PowerSystem{AC}(
        name = "IEEE 14-Bus",
        base_mva = 100.0,
    )
    
    # Buses
    push!(sys.buses, Bus{AC}(index=1, name="Bus 1", bus_type=REF_BUS, ...))
    push!(sys.buses, Bus{AC}(index=2, name="Bus 2", bus_type=PV_BUS, ...))
    # ...
    
    # Generators
    push!(sys.generators, Generator(index=1, bus=1, pg_mw=232.4, ...))
    # ...
    
    # Branches
    push!(sys.branches, Branch{AC}(index=1, from_bus=1, to_bus=2, ...))
    # ...
    
    return sys
end
```

## CSV Export

### `export_csv`

Export component data to CSV files.

```julia
export_csv(directory::String, sys::PowerSystem{K})
export_csv(directory::String, jpc::PowerCaseData{K})
```

**Generated Files:**
- `buses.csv`
- `generators.csv`
- `branches.csv`
- `loads.csv`
- `converters.csv`

**Example:**
```julia
sys = case14()
export_csv("./output/", sys)
```

### CSV Format

**buses.csv:**
```csv
index,name,bus_type,vm_pu,va_deg,pd_mw,qd_mvar,vmax_pu,vmin_pu
1,Bus 1,REF_BUS,1.05,0.0,0.0,0.0,1.1,0.9
2,Bus 2,PV_BUS,1.045,0.0,21.7,12.7,1.1,0.9
```

**branches.csv:**
```csv
index,from_bus,to_bus,r_pu,x_pu,b_pu,rate_a_mva,in_service
1,1,2,0.01938,0.05917,0.0528,100.0,true
2,1,5,0.05403,0.22304,0.0492,100.0,true
```

## ETAP Import

### `load_etap_csv`

Import system from ETAP CSV export format.

```julia
load_etap_csv(bus_file::String, branch_file::String; load_file=nothing) -> PowerSystem{AC}
```

**Parameters:**
- `bus_file` — Path to ETAP bus CSV
- `branch_file` — Path to ETAP branch CSV
- `load_file` — Optional path to load CSV

**Example:**
```julia
sys = load_etap_csv(
    "etap_buses.csv",
    "etap_branches.csv",
    load_file="etap_loads.csv"
)
```

### ETAP CSV Format

**Buses:**
```csv
BusID,Name,kV,Type,Vm,Va
1,Main Bus,13.8,Swing,1.0,0.0
2,Load Bus,13.8,Load,1.0,0.0
```

**Branches:**
```csv
ID,FromBus,ToBus,R_pu,X_pu,Length_km
1,1,2,0.01,0.05,5.0
```

## Data Conversion

### `to_matrix`

Convert struct-based system to matrix format.

```julia
to_matrix(sys::PowerSystem{K}) -> PowerCaseData{K}
to_matrix(hybrid::HybridPowerSystem) -> HybridPowerCaseData
```

### `from_matrix`

Convert matrix format to struct-based system.

```julia
from_matrix(jpc::PowerCaseData{K}) -> PowerSystem{K}
from_matrix(hjpc::HybridPowerCaseData) -> HybridPowerSystem
```

### Round-Trip Example

```julia
# Start with struct
sys = case14()

# Convert to matrix
jpc = to_matrix(sys)

# Modify in matrix form
jpc.bus[5, :PD] *= 1.2

# Convert back
sys2 = from_matrix(jpc)

# Verify
@assert nbuses(sys) == nbuses(sys2)
```

## Validation

### `validate_case`

Validate power system data for common errors.

```julia
validate_case(sys::PowerSystem{K}) -> Bool
validate_case(jpc::PowerCaseData{K}) -> Bool
```

**Checks:**
- Bus numbers are unique and positive
- Branch buses exist
- Generator buses exist
- At least one reference bus
- Voltage limits are valid
- Power limits are valid

### `is_valid`

Quick validity check.

```julia
is_valid(sys::PowerSystem{K}) -> Bool
```

### `has_warnings`

Check for non-fatal data issues.

```julia
has_warnings(sys::PowerSystem{K}) -> Bool
```

**Example:**
```julia
sys = case14()

if validate_case(sys)
    println("System is valid")
else
    println("Validation failed")
end

if has_warnings(sys)
    println("System has warnings")
end
```

## Best Practices

### File Organization

```
project/
├── data/
│   ├── matpower/
│   │   ├── case14.m
│   │   └── case118.m
│   ├── json/
│   │   └── modified_case.json
│   └── csv/
│       ├── buses.csv
│       └── branches.csv
└── results/
    └── output.csv
```

### Import Workflow

```julia
# 1. Load from file
jpc = load_matpower("data/matpower/case14.m")

# 2. Validate
if !validate_case(jpc)
    error("Invalid case data")
end

# 3. Convert to struct for manipulation
sys = from_matrix(jpc)

# 4. Modify
for gen in sys.generators
    gen.pmax_mw *= 1.1  # Increase capacity
end

# 5. Save modified version
save_json("data/json/modified_case.json", sys)
```

### Export Workflow

```julia
# 1. Build or modify system
sys = case14()
sys.name = "Modified IEEE 14-Bus"

# 2. Export to multiple formats
save_julia_case("output/case14_modified.jl", sys)
save_json("output/case14_modified.json", sys)
export_csv("output/csv/", sys)

# 3. Print summary
print_summary(sys)
```
